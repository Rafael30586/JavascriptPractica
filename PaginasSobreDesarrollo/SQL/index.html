Ejemplos de querys

SELECT * FROM tabla;

Muestra la tabla completa.

SELECT nombre FROM personas;

Muestra el nombre de todas las personas de la tabla.

SELECT marca, modelo FROM computadoras WHERE precio = 100000;

Muestra la marca y el modelo de las computadoras con un precio de 100000.

Tipos de datos

VARCHAR(tamaño): cadenas de caracteres;
el tamaño puede ser desde 0 hasta 65535.

INT(tamaño): números enteros;
el tamaño puede ser desde -9223372036854775808 a 9223372036854775807.

DECIMAL(tamaño,d): números enteros;
el tamaño puede ser desde -9223372036854775808 a 9223372036854775807;
la d como parámetro determina la cantidad de dígitos después de la coma.
