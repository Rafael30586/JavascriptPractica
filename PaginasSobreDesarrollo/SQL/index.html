SQL

Ejemplos de querys

SELECT * FROM tabla

Muestra todos los datos de una tabla

SELECT nombre FROM personas

Muestra el nombre de todas las filas de la tabla "personas"

SELECT modelo, marca FROM laptops WHERE precio = 1000

Muestra el modelo y la marca de todas las laptops donde el precio es 1000

Tipos de datos

VARCHAR(tamaño): cadenas de caracteres; el tamaño puede ser desde 0 hasta 65535.

INT(tamaño): números enteros; el tamaño puede ser desde -9223372036854775808 a 9223372036854775807.

DECIMAL(tamaño,d): números enteros;
el tamaño puede ser desde -9223372036854775808 a 9223372036854775807;
la d como parámetro determina la cantidad de dígitos después de la coma.
