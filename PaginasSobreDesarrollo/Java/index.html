Java

Tipos de datos

byte: tamaño: 1 byte;
Guarda números con valores que van desde -128 hasta 127.

short: tamaño: 2 bytes;
Guarda números con valores que van desde -32,768 hasta 32,767.

int: tamaño: 4 bytes;
Guarda números con valores que van desde -2,147,483,648 hasta 2,147,483,647.

long: tamaño: 8 bytes;
Guarda números con valores que van desde -9,223,372,036,854,775,808 hasta 9,223,372,036,854,775,807.

float: tamaño: 4 byte;
Guarda números fraccionarios que pueden tener entre 6 y 7 dígitos decimales.

double: tamaño: 8 byte;
Guarda números fraccionarios que pueden tener 15 dígitos decimales.

boolean: tamaño: 1 byte;
Guarda valores de verdadero o falso (true o false).

char: tamaño: 2 byte;
Guarda caracteres individuales, principalmente letras.
