Sobre mí

Hola, soy Rafael Alvarez, tengo más de dos años de experiencia trabajando en proyectos personales como desarrollador de software utilizando diferentes lenguajes y tecnologías como java, spring boot, SQL, spring security, docker, git, html, css, javascript, etc. Actualmente mi objetivo es trabajar como desarrollador web backend para empresas. Durante estos más de dos años no solamente me he dedicado a programar de forma independiente, sino que también he realizado mentorías durante el cursado de Argentina programa y, antes de trabajar como desarrollador, me he desempeñado en el área de la docencia, lo cual me ha permitido mejorar mis habilidades blandas, que son fundamentales para el trabajo en equipo. Proyectos

API sobre personajes de ficción con registro de usuarios

API desarrollada con spring boot y spring security que permite añadir personajes de ficcion a una base de datos SQL. Las acciones a realizar dependeran del rol del usuario, ejemplo: un ADMIN tendra la posiblidad de realizar acciones que un usuario comun no puede.

Aplicación fullstack

Versión Fullstack de la aplicación sobre personajes de ficción, aunque, en este caso, no no posee registro de usuarios en la base de datos. Aprovecha el motor de plantillas thymeleaf para la visualizción de listas de personajes y para verlos individualmente con su respectiva foto.
