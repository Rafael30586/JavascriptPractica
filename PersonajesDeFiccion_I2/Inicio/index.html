Si es tu primera vez utilizando esta aplicación deberás agregar obras. Para esto tendrás que presionar el botón que corresponda en esta página. Una obra puede ser un video juego, una película, una serie, etc. Una vez estas obras estén almacenadas en la base de datos podrás agregar los personajes de ficción que más te gusten. Ejemplo: Si querés agregar al personaje Superman primero tendrás que almacenar en la base de datos una película, video juego, serie o comic que contenga a este personaje como pueden ser Man of Steel, Injustice 2, etc
